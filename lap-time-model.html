Lap Time Prediction Model

Overview

This project develops a machine learning model to predict Formula 1 lap times based on race conditions and tyre usage. The model uses historical race data to understand how factors such as tyre compound, tyre age, and fuel load influence lap performance.

Key Concepts

  • Regressio modelling
  • Feature Engineering
  • Model Evaluation
  • Performance Prediction
