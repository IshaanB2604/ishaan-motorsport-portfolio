Race Strategy Simulator

Overview

This project simulates Formula 1 race strategies using tyre degradation modelling. The model evaluates different pit stop strategies to determine the fastest race plan.

Key Concepts

  • Tyre degradation modelling
  • Pit stop timing optimisation
  • Race simulation

Example Output

Graph showing lap time vs tyre age.

Technologies Used
